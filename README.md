# Job-match-Analyzer

This project analyzes how well a resume matches a job description using NLP techniques.

## Features
- Upload CV (PDF)
- Compare with job description
- Calculate text similarity score
- Identify matched and missing skills
- Provide recommendations

## Techniques Used
- TF-IDF Vectorization
- Cosine Similarity
- Text preprocessing (cleaning + PDF extraction)

## Tools
- Python
- Scikit-learn
- PDF processing libraries

## How it works
The system extracts text from a resume, compares it with a job description, 
and evaluates how well the candidate matches the job requirements.
